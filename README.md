# Quiz Game (Java Console App)

A simple console-based quiz game built using Java.  
This project demonstrates core programming concepts like arrays, loops, user input handling, and conditional logic.

---

## Features

- Multiple-choice quiz (A, B, C, D)
- Real-time answer validation
- Score tracking
- Replay option after completion
- Input validation for user responses

---

## Tech Stack

- Language: Java  
- Concepts Used:
  - Arrays and 2D Arrays
  - Loops (for, while)
  - Conditional statements (if-else)
  - User input using Scanner

---

## Project Structure

quizGame.java   # Main Java file containing the quiz logic

---

## How to Run

1. Clone the repository
   git clone https://github.com/your-username/quiz-game-java.git
   cd quiz-game-java

2. Compile the program
   javac quizGame.java

3. Run the program
   java quizGame

---

## How It Works

1. The program displays a question.
2. The user selects an answer (A / B / C / D).
3. The program validates the input and checks correctness.
4. The score is updated accordingly.
5. At the end, the user can choose to replay the quiz.
